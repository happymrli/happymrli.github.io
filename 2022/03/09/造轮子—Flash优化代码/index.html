✔优化代码技巧：

  • 串口监视器输出:Serial.println(“……”);[SRAM] → Serial.println(F(“……”));[Flash]
  • 常量：const int a = 125;[SRAM] → const int PROGMEM a = 125;[Flash]
  • 变量（尽量使用局部变量代替全局变量）：全局变量[独享SRAM] → 局部变量[共享SRAM]
  • 数据类型：int [2字节] → bool/byte[1字节]
